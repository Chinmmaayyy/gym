# KEY FITNESS — The Unisex Gym

A premium, production-ready website for **Key Fitness**, built to the specification in the
[Design Bible](./docsa) (Parts 1–13). Dark luxury aesthetic · black / cyan / yellow / white identity ·
conversion-focused.

> Transform Your Body. Transform Your Life.

---

## Tech Stack

| Layer        | Choice                                            |
| ------------ | ------------------------------------------------- |
| Framework    | Next.js 16 (App Router) · React 19                |
| Language     | TypeScript (strict)                               |
| Styling      | Tailwind CSS v4 (CSS-first `@theme` design tokens)|
| Animation    | Framer Motion                                     |
| Icons        | Lucide React (+ inline brand glyphs)              |
| Forms        | React Hook Form + Zod                             |
| Fonts        | Bebas Neue (display) · Inter (body) via `next/font`|

## Design System

All tokens live in [`app/globals.css`](./app/globals.css) under `@theme` and are consumed as Tailwind
utilities (`bg-bg`, `text-primary`, `rounded-card`, …).

- **Colors** — `#050505` bg · `#2CC5F4` primary · `#FFD400` highlight (90 / 7 / 3 distribution)
- **Radius** — `rounded-btn` 14px · `rounded-card` 20px · `rounded-section` 24px · `rounded-img` 18px
- **Motion** — 0.5–0.8s, `cubic-bezier(0.22,1,0.36,1)`, animate-once on scroll, respects `prefers-reduced-motion`

## SEO & Performance

- Per-page metadata (unique title/description), Open Graph, Twitter cards, canonical URLs
- `sitemap.xml`, `robots.txt`, and PWA `manifest.webmanifest` generated by the App Router
- JSON-LD: `HealthClub` / `SportsActivityLocation` / `LocalBusiness` + `FAQPage`
- All routes prerender as static content; fonts and images are optimized; animations are GPU-friendly
