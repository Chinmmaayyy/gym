# KEY FITNESS --- Website Design Bible

## Part 2 --- Website Architecture & User Journey

# Primary Objective

The website should convert visitors into membership enquiries while
showcasing Key Fitness as a premium fitness destination.

------------------------------------------------------------------------

# Primary Navigation

1.  Home
2.  About
3.  Services
4.  Membership Plans
5.  Trainers
6.  Gallery
7.  Contact

CTA Button: **Join Now**

Sticky transparent navbar that becomes solid on scroll.

------------------------------------------------------------------------

# User Journey

## New Visitor

Landing Page → Services → Membership Plans → Gallery → Contact →
WhatsApp

## Returning Visitor

Landing Page → Membership Plans → Contact Form

## Mobile User

Hero → Join Now → Membership Plans → WhatsApp

------------------------------------------------------------------------

# Website Pages

## Home

Purpose: Create a strong first impression and encourage enquiries.

Sections:

1.  Hero
2.  Why Choose Key Fitness
3.  Services
4.  Membership Plans
5.  Personal Training
6.  Facilities
7.  Gallery Preview
8.  Testimonials
9.  FAQ
10. Contact CTA
11. Footer

------------------------------------------------------------------------

## About

Include:

-   Gym Story
-   Mission
-   Vision
-   Core Values
-   Modern Equipment
-   Certified Trainers
-   Brand Philosophy

------------------------------------------------------------------------

## Services

Cards for:

-   Gym Training
-   CrossFit
-   Cardio
-   Personal Training
-   Steam Bath
-   Massage

Each card should include:

-   Premium icon
-   Short description
-   CTA

------------------------------------------------------------------------

## Membership Plans

Display plans using premium pricing cards.

Features:

-   Highlight recommended plan
-   Compare benefits
-   Strong Join Now CTA

------------------------------------------------------------------------

## Trainers

Each trainer card should contain:

-   Professional Photo
-   Name
-   Specialization
-   Experience
-   Certifications
-   Social Links (optional)

------------------------------------------------------------------------

## Gallery

Use a masonry layout.

Categories:

-   Gym Interior
-   Equipment
-   Cardio
-   CrossFit
-   Personal Training
-   Members

Support lightbox preview.

------------------------------------------------------------------------

## Contact

Include:

-   Contact Form
-   Phone Number
-   WhatsApp Button
-   Email
-   Google Maps
-   Opening Hours

------------------------------------------------------------------------

# Footer

Columns:

-   Quick Links
-   Membership
-   Services
-   Contact
-   Social Media

Bottom Bar:

-   Copyright
-   Privacy Policy
-   Terms & Conditions

------------------------------------------------------------------------

# Conversion Strategy

Every major section should include a clear call-to-action.

Examples:

-   Join Today
-   Book Free Trial
-   View Plans
-   Contact Us
-   WhatsApp Now

------------------------------------------------------------------------

# Future Scalability

The architecture should allow future additions without redesign:

-   Multi-branch support
-   Online Membership Purchase
-   Payment Gateway
-   Admin Dashboard
-   Member Dashboard
-   Attendance Tracking
-   Workout Plans
-   Nutrition Plans
-   Blog
-   Events

------------------------------------------------------------------------

# Information Hierarchy

Priority:

1.  Hero
2.  Membership Plans
3.  Services
4.  Trust Indicators
5.  Gallery
6.  Testimonials
7.  Contact

Important information should always appear above the fold where
possible.

------------------------------------------------------------------------

# Performance Goals

-   Mobile-first
-   Fast loading
-   Lazy-loaded images
-   Optimized assets
-   SEO-friendly URLs
-   Accessible navigation
-   Smooth scrolling
-   Responsive across all devices

------------------------------------------------------------------------

# Final Objective

The user should be able to understand what Key Fitness offers, trust the
brand, view pricing, and contact the gym within a few minutes,
regardless of device.
