# KEY FITNESS --- Website Design Bible

## Part 1 --- Project Vision & Brand Identity

### Project Name

**KEY FITNESS**

**Tagline:** THE UNISEX GYM

------------------------------------------------------------------------

# Project Goal

Create India's most premium-looking gym website.

The website should communicate: - Premium - Luxury - Strength -
Discipline - Trust - Performance - Professionalism

Visitors should immediately feel: \> "This isn't an ordinary local gym."

Inspired by the quality of Nike, Gymshark, Equinox, Gold's Gym, Anytime
Fitness, and Apple.

# Target Audience

-   Age: 18--40
-   Students
-   Working Professionals
-   Fitness Enthusiasts
-   Athletes
-   Beginners
-   Weight Loss Clients
-   Muscle Gain Clients
-   CrossFit Enthusiasts
-   Personal Training Clients

# Brand Personality

-   Powerful
-   Bold
-   Premium
-   Modern
-   Minimal
-   Luxurious
-   Energetic
-   Confident
-   Professional

Avoid clutter, generic layouts, childish visuals, or excessive colors.

# Design Philosophy

Less is More.

Use: - Large whitespace - Strong typography - Cinematic imagery - Smooth
animations - Consistent spacing

# Visual Direction

Dark luxury aesthetic with: - Black backgrounds - Cyan accents - Yellow
highlights - Glassmorphism - Rounded cards - Premium photography

# Color Palette

  Usage                  Color
  ---------------------- ---------
  Primary Background     #050505
  Secondary Background   #0E0E0E
  Card Background        #111111
  Border                 #242424
  Primary Accent         #2CC5F4
  Secondary Accent       #17A7E3
  Highlight              #FFD400
  Text Primary           #FFFFFF
  Text Secondary         #C9C9C9
  Muted Text             #8D8D8D

Color distribution: - 90% Black - 7% Cyan - 3% Yellow

# Typography

Headings: - Bebas Neue - Anton - League Spartan

Body: - Inter - Manrope - Plus Jakarta Sans

Desktop: - Hero: 72px - Heading: 48px - Body: 18px

Mobile: - Hero: 42px - Heading: 32px - Body: 16px

# UI Style

Modern SaaS meets Luxury Fitness.

Include: - Glassmorphism - Floating cards - Hover animations - Gradient
borders - Soft cyan glow

# Imagery

Use only premium fitness photography: - Gym interiors - Equipment -
CrossFit - Cardio - Personal training - High contrast - Dark lighting

# Spacing

-   Section spacing: 140px
-   Card gap: 32px
-   Padding: 24px

# Final Goal

Deliver a premium website that feels like an international fitness brand
while remaining true to Key Fitness's black, cyan, yellow, and white
identity.
