# KEY FITNESS --- Website Design Bible

# Part 3 --- Home Page Specification

## Goal

The homepage must immediately establish Key Fitness as a premium fitness
destination and guide visitors toward joining the gym.

------------------------------------------------------------------------

# Hero Section

## Layout

-   Full-screen (100vh)
-   Split layout on desktop
-   Single-column on mobile
-   Sticky transparent navbar

## Background

Use a cinematic fitness video or dark premium gym image with a
black-to-cyan overlay.

## Headline

**Transform Your Body. Transform Your Life.**

## Supporting Text

Train in a premium fully air-conditioned unisex gym with certified
trainers, CrossFit, cardio, personal training, steam bath, and massage
facilities.

## Primary CTA

-   Join Now

## Secondary CTA

-   View Membership Plans

## Hero Stats

-   Certified Trainers
-   Fully AC Gym
-   Modern Equipment
-   Personal Training
-   CrossFit

Animate counters on scroll.

# Animation Specification

Use Framer Motion.

Animations:

-   Fade Up
-   Fade Left
-   Fade Right
-   Stagger Children
-   Parallax Hero
-   Number Counter
-   Card Hover Lift
-   Button Ripple
-   Smooth Scroll
-   Image Reveal

Duration: 0.5--0.8 seconds.

Use subtle easing.

------------------------------------------------------------------------

# Responsive Rules

Desktop: Maximum content width 1440px.

Tablet: Two-column layouts.

Mobile: Single-column layouts with full-width CTAs.

------------------------------------------------------------------------

# Accessibility

-   High contrast
-   Keyboard navigation
-   Alt text
-   Proper heading hierarchy
-   Visible focus states

------------------------------------------------------------------------

# Performance

-   Lazy load images
-   Compress media
-   Next.js Image optimization
-   Lighthouse target: 95+

------------------------------------------------------------------------

# Success Criteria

Within the first 10 seconds, a visitor should know: - What Key Fitness
offers - Why it's different - Membership options - How to contact the
gym - How to join
