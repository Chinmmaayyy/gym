# Motion Guidelines

Use Framer Motion.

Duration: 0.5--0.8 sec

Ease: easeOut

Never over-animate.

Prioritize elegance.

------------------------------------------------------------------------

# Accessibility

Minimum contrast ratio.

Keyboard focus.

Screen-reader labels.

Large clickable targets.

------------------------------------------------------------------------

# Design Consistency Rules

-   Use consistent spacing everywhere.
-   Never mix border radius values.
-   Every card follows the same visual language.
-   Every CTA uses the primary button style.
-   Icons remain consistent throughout the site.
-   All components should be reusable and scalable.

------------------------------------------------------------------------

# Final Goal

This design system should allow new pages and future features to be
added without changing the visual identity of Key Fitness.
