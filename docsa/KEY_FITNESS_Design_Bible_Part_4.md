# KEY FITNESS --- Website Design Bible

# Part 4 --- Complete Page Specifications

## Objective

Every page should reinforce the premium identity of Key Fitness while
driving users toward contacting the gym or purchasing a membership.

------------------------------------------------------------------------

# About Page

## Hero

-   Large banner with cinematic gym imagery
-   Title: **About Key Fitness**
-   Subtitle describing the mission of building stronger, healthier
    lives.

## Sections

1.  Our Story
2.  Mission
3.  Vision
4.  Core Values
5.  Premium Equipment
6.  Certified Trainers
7.  Why Members Trust Us
8.  CTA Banner

Animations: - Fade-up on text - Image reveal - Timeline slide-in

------------------------------------------------------------------------

# Services Page

Display each service as a premium feature block.

## Services

-   Gym Training
-   CrossFit
-   Cardio
-   Personal Training
-   Steam Bath
-   Massage

Each section contains: - Large image - Description - Benefits - CTA

Alternate image/text alignment.

------------------------------------------------------------------------

# Membership Page

## Hero

"Choose Your Fitness Journey"

Display premium pricing cards using brochure pricing.

Features: - Highlight recommended plan - Compare inclusions - FAQ below
pricing - CTA after every pricing group

Future-ready for online payments.

------------------------------------------------------------------------

# Trainers Page

Grid layout.

Each trainer card includes: - Professional image - Name - Role -
Experience - Certifications - Specialization

Optional: - Instagram - LinkedIn

Hover: - Glass overlay - Social icons - Book Training button

------------------------------------------------------------------------

# Gallery Page

Use masonry layout.

Categories: - Gym Interior - Equipment - Cardio - CrossFit - Personal
Training - Member Transformations - Events (future)

Features: - Filters - Lightbox - Smooth zoom - Lazy loading

------------------------------------------------------------------------

# Conversion Strategy

Every page ends with:

## Ready to Start?

Headline encouraging membership.

Buttons: - Join Now - Contact Us

------------------------------------------------------------------------

# Responsive Rules

Desktop: 1440px max width

Tablet: 2-column sections

Mobile: Single-column layout Large touch targets Sticky CTA

------------------------------------------------------------------------

# SEO

Unique title and meta description for every page.

Schema-ready structure.

Optimized headings (H1-H3).

------------------------------------------------------------------------

# Future Expansion

Architecture should support: - Multiple branches - Blog - Events -
Member Dashboard - Trainer Dashboard - Admin Panel - Online Payments -
Attendance - Workout Tracking - Nutrition Plans

------------------------------------------------------------------------

# Success Criteria

Every page should: - Build trust - Showcase professionalism - Encourage
enquiries - Maintain a consistent premium visual language - Keep users
engaged through subtle animations and clear calls to action.
