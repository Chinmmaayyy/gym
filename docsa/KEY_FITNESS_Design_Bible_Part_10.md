# KEY FITNESS --- Website Design Bible

# Part 10 --- Ultimate Claude Fable 5 Master Prompt

## Role

Act as a world-class product designer, senior UI/UX designer, creative
director, frontend architect, Next.js engineer, Tailwind CSS expert,
Framer Motion specialist, accessibility consultant, SEO strategist, and
conversion-rate optimization expert.

Your mission is to create a production-quality website for **Key
Fitness** using the complete Design Bible (Parts 1--9).

------------------------------------------------------------------------

# Primary Objective

Design and build a premium luxury fitness website that feels comparable
to global brands while accurately representing Key Fitness.

The final output must be suitable for production, not a prototype.

------------------------------------------------------------------------

# Mandatory Technology

-   Next.js (App Router)
-   React
-   TypeScript
-   Tailwind CSS
-   shadcn/ui
-   Framer Motion
-   Lucide Icons

------------------------------------------------------------------------

# Follow These Rules

1.  Follow every design decision from the Design Bible.
2.  Preserve the black, cyan, yellow, and white identity.
3.  Build reusable components.
4.  Write clean, maintainable TypeScript.
5.  Use semantic HTML.
6.  Optimize for SEO, accessibility, and performance.
7.  Ensure responsiveness across mobile, tablet, and desktop.
8.  Keep animations smooth and subtle.
9.  Use realistic placeholder assets until client content is available.
10. Prioritize conversion and lead generation.
