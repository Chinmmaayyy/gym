# KEY FITNESS --- Website Design Bible

# Part 11 --- Visual Direction Bible

## Objective

Define the complete visual language so every screen feels like it
belongs to one premium fitness brand.

------------------------------------------------------------------------

# Brand Inspiration

Do **not** copy any existing website.

Take inspiration from the design quality and user experience of:

-   Apple
-   Nike
-   Gymshark
-   Equinox
-   Porsche
-   Tesla

Focus on premium execution rather than imitation.

------------------------------------------------------------------------

# Overall Mood

Keywords:

-   Powerful
-   Premium
-   Luxury
-   Energetic
-   Clean
-   Bold
-   Confident
-   Minimal
-   High Performance

Visitors should immediately think:

> "This gym looks world-class."

------------------------------------------------------------------------

# Art Direction

## Lighting

-   Cinematic
-   High contrast
-   Soft blue rim lighting
-   Deep blacks
-   Premium reflections

Avoid flat lighting.

------------------------------------------------------------------------

# Photography Style

Use only high-quality imagery featuring:

-   Modern gym interiors
-   Premium equipment
-   People actively training
-   Strength training
-   Functional fitness
-   CrossFit
-   Cardio

Avoid:

-   Low-resolution photos
-   Cartoon illustrations
-   Obvious stock-photo poses
-   Cluttered backgrounds

------------------------------------------------------------------------

# Color Psychology

## Black

Represents: - Luxury - Strength - Confidence

## Cyan

Represents: - Energy - Modern technology - Freshness

## Yellow

Represents: - Achievement - Attention - Success

Use yellow only for emphasis.

------------------------------------------------------------------------

# Visual Hierarchy

Priority order:

1.  Hero headline
2.  CTA
3.  Membership plans
4.  Services
5.  Trust indicators
6.  Testimonials
7.  Contact

Every page should have one obvious primary action.

------------------------------------------------------------------------

# Layout Philosophy

-   Large whitespace
-   Modular sections
-   Consistent spacing
-   Strong alignment
-   Minimal distractions

Never overcrowd a page.

------------------------------------------------------------------------

# Glassmorphism

Cards should include:

-   Semi-transparent background
-   Soft blur
-   Thin border
-   Subtle shadow

Never overuse glass effects.

------------------------------------------------------------------------

# Imagery Placement

Hero: Large edge-to-edge visual.

Content: Alternate image and text sections.

Gallery: Balanced masonry grid.

------------------------------------------------------------------------

# Iconography

Use Lucide icons only.

Style:

-   Outline
-   Clean
-   Consistent stroke width
-   Modern

------------------------------------------------------------------------

# Buttons

Primary: Cyan gradient

Secondary: Transparent with white border

Hover: Lift + glow

------------------------------------------------------------------------

# Cards

Every card should share:

-   Rounded corners
-   Soft shadow
-   Consistent padding
-   Hover elevation
-   Premium spacing

------------------------------------------------------------------------

# Background Effects

Allow subtle:

-   Gradient overlays
-   Mesh gradients
-   Noise texture (very light)
-   Ambient glow

Avoid distracting backgrounds.

------------------------------------------------------------------------

# Motion Language

Animations should feel:

-   Elegant
-   Smooth
-   Intentional
-   Fast

Avoid playful or exaggerated effects.

------------------------------------------------------------------------

# Section Rhythm

Alternate between:

-   Dark background
-   Slightly lighter surface
-   Full-width imagery
-   Spacious content

This creates visual flow without monotony.

------------------------------------------------------------------------

# Trust Indicators

Highlight:

-   Certified Trainers
-   Fully AC Gym
-   Modern Equipment
-   Personal Training
-   CrossFit
-   Steam Bath
-   Massage

Display with premium iconography.

------------------------------------------------------------------------

# Visual Consistency Rules

-   One design language
-   One spacing system
-   One typography scale
-   One animation style
-   One color system

No component should appear visually unrelated.

------------------------------------------------------------------------

# Quality Benchmark

The finished website should look like it was designed by a premium
digital agency for a national fitness chain rather than a local
business.

Every screen should be presentation-worthy and suitable for a
professional portfolio.
